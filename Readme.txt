Codigos
"pwd" -> indica dónde estás posicionado con git bush
"cd" -> para ingresar a la dirección, ej cd "/c/Users/ZAMIRA_MONTORO"
"git status" -> sirve para ver el estado del proyecto
"git add ." -> agrega todos los archivos modificados
"git commit -m "indicamos los códigos a utilizar en git bush" " -> guarda los cambios localmente
"git push origin master" -> sube los cambios a GitHub
